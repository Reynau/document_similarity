Compilar con el comando 'make'
Borrar archivos generados con el comando 'make file'

El programa hay que ejecutar con el archivo main que generará el Makefile y aceptará como parámetros la ruta de dos documentos para comparar.
Por ejemplo 'main doc1.txt doc1.txt' para dos documentos llamados 'doc1.txt' y 'doc2.txt' ubicados en la raiz del proyecto,
o por ejemplo main randomDocs/doc1.txt randomDocs/doc1.txt' para dos documentos ubicados en la carpeta './randomDocs' en la raiz del proyecto.
